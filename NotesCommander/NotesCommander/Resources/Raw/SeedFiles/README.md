# Демонстрационные данные для NotesCommander

Эта папка содержит начальные демонстрационные данные для приложения NotesCommander.

## Структура

```
SeedFiles/
├── audio/
│   └── demo-note.wav          # Демонстрационный аудиофайл (речь JFK)
└── photos/
    ├── demo-photo.jpg         # Демонстрационная фотография (опционально)
    └── README.md              # Инструкция по добавлению фото
```

## Автоматическая инициализация

При первом запуске приложения:
1. Проверяется флаг `is_seeded` в настройках приложения
2. Если данные еще не инициализированы, загружается `SeedData.json`
3. Создается демонстрационная заметка с:
   - Заголовком: "Добро пожаловать в NotesCommander"
   - Аудиофайлом из речи JFK (~11 секунд)
   - Распознанным текстом из речи
   - Категорией "Идеи"
   - Тегами "демо" и "история"
   - Опциональной фотографией (если файл `demo-photo.jpg` существует)

## Как добавить демо-фотографию

1. Поместите файл с именем `demo-photo.jpg` в папку `photos/`
2. Рекомендуемый размер: до 1024x1024 пикселей
3. Формат: JPG

## Сброс данных

Для повторной инициализации демонстрационных данных:
1. Откройте страницу "Настройки" в приложении
2. Используйте функцию "Сброс данных"
3. Это очистит все таблицы и загрузит заново данные из `SeedData.json`

## Технические детали

- Аудиофайлы и фотографии копируются в `FileSystem.AppDataDirectory` при инициализации
- JSON-файл содержит также определения категорий и тегов
- Файл `SeedData.json` расположен в `Resources/Raw/` и автоматически включается в пакет приложения
